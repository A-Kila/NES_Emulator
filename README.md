# NES Emulator
NES Emulator written in c++ 

## How to Run
Find `nes_emulator` binary in `build/src/` folder  
Run `./nes_emulator PATH_TO_CARTRIDGE_ROM`

### Controls
`w a s d` -> arrow keys  
`j`&emsp;&emsp;&emsp; -> A button  
`k`&emsp;&emsp;&emsp; -> B button  
`Enter`&emsp; -> START button  
`Esc`&emsp;&emsp; -> SELECT button

### Requirements
* CMake version minimum 3.30.4
* C++ 23

### Dependencies
* SDL

### Mappers Implemented
* nrom mapper (mapper 0)
